# SRT Translator

Autonomous translation tool for SRT subtitle files.

## How to translate subtitles

The `translate-srt` skill is invoked automatically when you ask to translate subtitles.

## Structure

- `.claude/skills/translate-srt/` - Translation skill
- `hooks/` - Split, join and validation scripts
- `input/` - SRT files to translate
- `output/` - Translated files
